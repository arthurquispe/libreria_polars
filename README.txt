Este es un proyecto recopilatorio y práctico que tiene la finalidad de crear colaborativamente un tutorial de la librería Polars para Python. 
En él, explicamos de qué va esta librería, la comparamos con Pandas y otras librerías cuyo objetivo es la manipulación de datos.
Este tutorial está principalmente orientado a programadores junior o personas que quieran iniciarse en ciencia de datos.

El eje creativo de este tutorial se centra en la rivalidad que existe entre la librería Pandas y Polars. 
Por ello, se hacen símiles entre el programa de televisión "Escandalosos" y las librerías. 
Esto se hizo con la finalidad de amenizar la lectura de este tutorial. Esperamos que les sea útil.

¿Interesado en probar Polars? Aquí en GitHub te dejamos los archivos csv. de este tutorial para que te inicies en Polars.
